# react-native-android-ios-web-ts-template

A small React Native template supporting Android, iOS & Web including the following libraries:

- react-native-paper
- react-native-vector-icons
- react-navigation
- react-native-web
- webpack

## Install dependecies
`npm install`

`cd ios && pod install && cd ..`

## Run
Run Android:
`npm run android`

Run iOS: 
`npm run ios`

Run Web:
`npm run web`
